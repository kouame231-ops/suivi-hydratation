\## 🚀 Guide d'utilisation pour le test en local



Pour ouvrir en localr et tester ce projet sur votre machine via \*\*Android Studio Meerkat Feature Drop\*\* :



\## 1. Cloner le depot ou télécharger le fichier ZIP

Clonez ce projet sur votre machine depuis le depot GitHub :

\# pour cloner taper:

```bash

git clone \[https://github.com/kouame231-ops/suivi-hydratation.git](https://github.com/kouame231-ops/suivi-hydratation.git)

```

\# ou téléchargez format ZIP "suivi-hydratation" inclu dans le depot GitHub et decompresser 



\## 2. Ouvrir le projet dans Android Studio

Lancez Android Studio Meerkat Feature Drop.



Sélectionnez Open (Ouvrir) et naviguez jusqu'au dossier du projet cloné.



Laissez Android Studio synchroniser le projet avec Gradle (cette étape peut prendre quelques minutes lors de la première ouverture).



\## 3. Lancer l'application

Branchez un appareil Android physique (avec le débogage USB activé) ou lancez un Émulateur Android (AVD).



Cliquez sur le bouton ▶️ Run (Exécuter) dans la barre d'outils d'Android Studio pour compiler et lancer l'application.
